# 📌 About

Ashirwad – Organizer's Assistant is an **AI-enabled Community Management Platform** designed to simplify the day-to-day operations of community organizations.

It helps committees manage members, registrations, donations, volunteers, and events through a simple, modern, mobile-friendly platform.

# 🌺 Founding Beta Customer

## Arohan – A Socio Cultural Forum

📍 Kadugodi, Bengaluru

Arohan is the **first paid Founding Beta Customer** of Ashirwad.

This deployment serves as the reference implementation for future community organizations.
